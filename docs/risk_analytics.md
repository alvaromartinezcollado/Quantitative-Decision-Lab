# Cross-Asset Risk Analytics

## Objective

The objective of this module is to compute formal risk and return metrics for the selected asset universe.

This phase uses the cleaned adjusted close prices and daily returns generated by the Market Data Pipeline.

## Metrics Computed

### Annualized Return

Annualized return is estimated by multiplying the average daily return by the number of trading days in a year.

### CAGR

CAGR, or Compound Annual Growth Rate, measures the annual compounded growth rate of an asset from the beginning to the end of the analysed period.

### Annualized Volatility

Annualized volatility measures the dispersion of daily returns scaled to a yearly horizon.

### Maximum Drawdown

Maximum Drawdown measures the worst historical decline from a previous peak.

### Sharpe Ratio

The Sharpe Ratio measures excess return per unit of total volatility.

### Sortino Ratio

The Sortino Ratio measures excess return per unit of downside volatility.

### Historical VaR

Historical Value at Risk estimates a loss threshold at a given confidence level based on historical returns.

### Historical CVaR

Historical Conditional Value at Risk measures the average loss beyond the VaR threshold.

## Interpretation

This module allows assets to be compared not only by return, but also by risk and risk-adjusted performance.

An asset with high historical return is not necessarily the best decision if it also has high volatility, severe drawdowns or poor downside risk.

## Outputs

This module produces:

- annualized return;
- CAGR;
- annualized volatility;
- maximum drawdown;
- Sharpe Ratio;
- Sortino Ratio;
- Historical VaR 95%;
- Historical CVaR 95%;
- a consolidated risk summary table.
